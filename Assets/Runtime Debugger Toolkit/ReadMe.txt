Runtime Debugger Toolkit - v1.0.0

Thank you for downloading Runtime Debugger Toolkit !

Quick Start:
- Open the scene in Demo/Playground.unity
- Press "Tab" to unlock your mouth and interact with the asset prefab.

Need help?
- Documentation: MyAsset/Documentation/Debugger Tool Manual.pdf
- Contact: ThornyDevStudio@gmail.com

/!\ For the demo you need to have cinemachine v2.10.5 and not higher otherwise it won't work directly.
